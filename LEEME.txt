Para compilar se puede ejecutar el archivo compile.sh, un script de Bash.
IMPORTANTE: El script necesita que curl esté instalado en el sistema para funcionar

Este script hace lo siguiente:
	- Dado que el programa está implementado en Kotlin, se instala el compilador de Kotlin. No requiere permisos de root, pero si necesitará realizar una descarga desde Internet. En caso de que en la máquina ya se disponga del compilador de Kotlin instalado, se pueden comentar estas líneas.
	- Posteriormente compila el proyecto, usando el comando kotlinc. Esto genera un fichero ejecutable por la Java Virtual Machine: TabooSearch.jar

Para ejecutar, hacer como para cualquier otro programa en Java:
	java -jar TabooSearch.jar distancias [aleatorios]
